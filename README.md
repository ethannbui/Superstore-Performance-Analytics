# Superstore-Performance-Analytics

**Introduction**

This project analyzes the SuperStore USA retail dataset (9,994 order records) to evaluate sales performance, customer behavior, product trends, and regional operations.

The dataset contains real transaction records that support:

- Inventory planning

- Regional performance analysis

- Profitability evaluation

**Project Goal**

To improve the operational and financial performance of SuperStore by analyzing sales, discount impact, and delivery performance.

**Tools**: Excel, MySQL, SISS and PowerBI
